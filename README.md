
The system works as a **thinking and writing layer** — you clip articles, drop them in raw, ingest, and the knowledge compounds. For any new project idea, you'd create a fresh vault, point [Synapse](https://github.com/angrysky56/project-synapse-mcp) at it, and the project knowledge grows organically from whatever sources and conversations you feed it. Code stays in its repo; the _why_ and _what we learned_ lives in the wiki.
