# AlphaEvolve

**Type:** Entity — AI system  
**Developer:** Google DeepMind (2025)

## What It Is

An AI coding agent that combines a large language model with evolutionary search to discover and optimize algorithms. It generated novel, verifiably correct algorithms by using Gemini to propose code mutations and an automated evaluator to select improvements — discovering algorithms that improved on decades-old human solutions for problems including matrix multiplication.

Notable as an example of AI-driven scientific discovery: the system autonomously found breakthroughs in areas where human experts had been stuck, directly relevant to the [[momoa-researcher]] and [[causal-state-edm-ood-isomorphism]] discussions of what constitutes machine-generated disruption vs. consolidation.
