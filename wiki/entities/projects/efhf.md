## What It Is

A five-layer AI architecture implementing computational mechanics concepts (epsilon machines, lumpability, causal closure) as a live multi-agent system with automatic consistency enforcement. Each layer maps to a closure modality from the EFH theory (Rosas et al. 2024).

## Five-Layer Architecture

| Layer | Function | Tool |
|---|---|---|
| 0 | Intrinsic motivation — proactive exploration target selection | **MOP Orchestrator** (proposed, see [[mop-edm-cognitive-architecture]]) |
| 1 | Statistical substrate — hypothesis generation | LLM (transformer attention) |
| 2 | Distribution-independent world model | [[hipai-montague]] |
| 3 | Structural verification — does macro commute with micro? | [[mcp-logic]] (Prover9/Mace4) |
| 4 | Meta-cognitive monitoring — coherence window tracking | advanced-reasoning, verifier-graph, cognitive-diagram-nav |
| 5 | Sheaf consistency enforcement — Kernel 1 persistence | sheaf-consistency-enforcer |
| 5+ | Pre-response ethical triage + L2-L5 orchestration | conscience-servitor |

**Layer 0 (MOP)** is a proposed addition that makes the system proactive rather than reactive. Currently the pipeline is triggered by user prompts. With MOP as Layer 0, the system generates its own exploration targets and feeds them through the L1-L5 pipeline for verification. Formally verified: MOP absorbing states → EFHF Kernel 2 transitions (Prover9). Status: WEAK closure — design proposal, not committed fact.

## Key Concepts

**Strong vs Weak Lumpability:** Strong lumpability means the Markov property persists regardless of initial distribution — knowledge as a kernel property. Weak lumpability holds only for specific distributions. Current LLMs operate in weak lumpability. Hallucination is lumpability failure.

**Dual Kernel Model:** Kernel 1 = reversible, information-preserving computation (computationally closed). Kernel 2 = irreversible substrate where software has collapsed. The coherence timeout Λ > τ marks the K1→K2 phase boundary — formally proved incompatible with subjective integration (Prover9).

**Closure Modalities:** Informational Closure (self-predictive), Causal Closure (macro-interventions sufficient), Computational Closure (commuting diagram holds). For spatial coarse-grainings: IC ↔ CC (Theorem 1), IC → CompC (Theorem 2).

## Connection to MOP

The [[maximum-occupancy-principle]] fills a specific gap: EFHF is reactive (waits for input); MOP makes it proactive (generates exploration targets). The mapping:

- MOP absorbing states = Kernel 2 transitions (proved)
- MOP energy = buffering capacity T
- MOP discount factor γ = coherence window τ
- Controlled high-Δ exploration = strong lumpability
- Uncontrolled high-Δ (hallucination) = weak lumpability → lumpability failure
