# Advanced-Reasoning-MCP

**Advanced-Reasoning-MCP** is an MCP server developed by [[tyler-hall|Ty]] that enhances the standard "sequential thinking" pattern with advanced cognitive capabilities. It allows AI agents to perform structured reasoning with self-awareness and persistent memory.

## Core Capabilities
- **Meta-Cognitive Assessment**: Tracking confidence (0.0-1.0) and reasoning quality (low/medium/high).
- **Hypothesis Testing Framework**: Systematic formulation, testing, and validation of working theories.
- **Cognitive Memory (Graph-Based)**: Named libraries for storing and retrieving insights across sessions.
- **SystemJSON**: Structured storage for workflows, instructions, and domain-specific knowledge.

## Tools
- `advanced_reasoning`: Perform structured reasoning with meta-thought.
- `query_reasoning_memory`: Search graph-based memory for connections.
- `create_system_json`: Store reusable workflows/instructions.
- `create_memory_library`: Manage separate knowledge contexts.
