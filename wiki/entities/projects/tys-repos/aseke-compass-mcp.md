# ASEKE-Compass-MCP

**ASEKE-Compass-MCP** is an MCP server developed by [[tyler-hall|Ty]] that provides systemic discernment by analyzing behavioral patterns. It maps biological "levers" (Panksepp primary emotional systems) to recognize external influence and internal psychological states.

## Key Features
- **Panksepp Mapping**: Maps situations/behaviors to the seven primary emotional systems (SEEKING, RAGE, FEAR, LUST, CARE, PANIC/GRIEF, PLAY).
- **Behavioral Signal Analysis**: Identifies patterns of algorithmic or leadership-based manipulation.
- **Political Orientation Bridge**: Assesses probable political tendencies based on active emotional systems.

## Tools
- `analyze_behavior`: Perform a structured ASEKE analysis.
- `match_patterns`: Match observed signals to behavioral archetypes.
- `bridge_to_political`: Assess political orientation tendencies.
