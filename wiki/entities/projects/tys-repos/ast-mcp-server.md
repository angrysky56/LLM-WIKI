# AST-MCP-Server

**AST-MCP-Server** is a tool developed by [[tyler-hall|Ty]] that transforms source code into a queryable Semantic Graph and a structured Abstract Syntax Tree (AST). It provides AI assistants with the spatial awareness needed to understand code structure beyond simple text.

## Features
- **Semantic Graph Construction**: Maps relationships between code entities (functions, classes, variables).
- **AST Traversal**: Provides granular access to code structure.
- **AI-Focused Analysis**: Optimized for providing context to LLM coding agents.
