# Emergent Functional Hierarchies Framework (EFHF)

The **Emergent Functional Hierarchies Framework (EFHF)** is a comprehensive architecture developed by [[tyler-hall|Ty]] that synthesizes computational mechanics, persistence theory, and cognitive architecture. It formalizes how macroscopic processes achieve "software status" through causal and computational closure.

## Five-Layer Architecture
1. **Layer 1 (Statistical Substrate)**: LLM hypothesis generation.
2. **Layer 2 (World Model)**: [[hipai-montague]] graph-based modeling.
3. **Layer 3 (Structural Verification)**: [[mcp-logic]] (Prover9/Mace4) consistency checks.
4. **Layer 4 (Meta-Cognitive Monitoring)**: [[advanced-reasoning-mcp]] confidence tracking.
5. **Layer 5 (Sheaf Consistency Enforcement)**: [[sheaf-consistency-enforcer]] monitoring lumpability.
6. **Layer 5+ (Ethical Triage)**: [[conscience-servitor]] for pre-response triage.

## Key Concepts
- **Closure Modalities**: Informational, Causal, and Computational closure.
- **Dual Kernel Model**: Kernel 1 (information-preserving) and Kernel 2 (irreversible hardware).
- **Strong vs. Weak Lumpability**: Distinguishes genuine closure from probabilistic hallucination.
