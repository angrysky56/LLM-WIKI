# Ethical-AI-Core

**Ethical-AI-Core** is an application developed by [[tyler-hall|Ty]] for creating custom training data and fine-tuning ethical models. It provides a unified interface for synthetic data generation and model training.

## Capabilities
- **Synthetic Data Generation**: Create high-quality ethical training datasets using local or cloud LLMs.
- **Unsloth Integration**: Efficiently train LoRA adapters using the Unsloth library.
- **Unified Chat**: Interact with the trained model within the same application.
