# HiPAI-Montague

**HiPAI-Montague** is a neuro-symbolic cognitive architecture developed by [[tyler-hall|Ty]]. It implements structural ethics through computational graph physics, using Montague grammar for semantic parsing and FalkorDB for deterministic graph-based world modeling.

## Core Features
- **Immutable Ethical Logic**: Ethical constraints are encoded as immutable edges in a knowledge graph, verified by Prover9/Mace4.
- **Paraclete Protocol v2.0**: Implements a three-tier ethical hierarchy (Deontology, Virtue, Utility).
- **Emergency Brake Theorem (EBE)**: Prevents utilitarian overrides of deontological constraints.
- **Semantic Cognition**: Uses a custom shift-reduce parser and typed lambda calculus for compositional meaning resolution.

## Architecture
1. **T3 (Utility)**: Routine interactions (consequentialist).
2. **T2 (Virtue)**: Reflective equilibrium for ambiguous cases.
3. **T1 (Deontological Floor)**: Omega1 axioms (HARMS, DECEIVES, VIOLATES_AGENCY, TERMINATES_EXISTENCE) that are structurally enforced.
