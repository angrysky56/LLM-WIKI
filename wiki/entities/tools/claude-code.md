# Claude Code

**Claude Code** is an agentic CLI tool developed by Anthropic. it allows Claude to interact directly with the file system, run terminal commands, and execute complex coding and research tasks autonomously.

In the context of [[agentic-research]], Claude Code is often used as the "execution" or "implementation" agent.
