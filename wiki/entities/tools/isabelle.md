# Isabelle

Isabelle is a generic proof assistant. It allows mathematical formulas to be expressed in a formal language and provides tools for proving those formulas in a logical calculus.

## Key Features
- **Isabelle/jEdit**: The primary Prover IDE.
- **Sledgehammer**: An interface for calling external automated theorem provers (ATPs).
- **Archive of Formal Proofs (AFP)**: A collection of proof libraries.
