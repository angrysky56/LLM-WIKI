# GoodRobot — Revenue Model and Financial Projections

> **Source:** Paperclip issue GOO-5 | **Status:** Completed | **Date:** 2026-05-12

---

## Pricing Strategy

### Subscription Tiers

| Tier | Price | Agent Seats | Features |
|------|-------|-------------|----------|
| **Starter** | $299/mo | 3 agents | Basic task routing, email/Slack notifications, 5 role templates, 1,000 tasks/mo |
| **Professional** | $899/mo | 10 agents | All Starter + API access, webhook integrations, unlimited tasks, priority support, 25 role templates |
| **Enterprise** | $2,499/mo | Unlimited | All Professional + SSO, audit logs, SLA guarantee, dedicated onboarding, custom role development |

### Per-Seat Agent Add-ons
- Additional agents beyond tier limit: $49/agent/mo (Professional+) or $29/agent/mo (Enterprise)
- API calls beyond plan limits: $0.002 per call

### Professional Services
- Custom integrations: $150–250/hr
- Onboarding and training: $2,000–5,000 flat fee
- Dedicated implementation manager: $3,000/mo

## Unit Economics

| Metric | Value |
|--------|-------|
| Average Revenue Per User (ARPU) | $650/mo |
| Customer Acquisition Cost (CAC) | ~$400–600 (digital channels) |
| LTV:CAC Ratio | >5x target |
| Payback Period | <12 months |

## Financial Projections

### Year 1
- 120 new customers
- ~$594,000 ARR (conservative)
- Total S&M Investment: ~$90,300
- LTV:CAC Ratio: 30x

### Year 2
- 400 total customers
- ~$2.2M ARR
- Path to break-even operational costs

## Revenue Diversification

1. **Subscription revenue** — core MRR from agent seat licenses
2. **Usage-based API fees** — scales with customer usage
3. **Professional services** — onboarding, custom integration
4. **Marketplace commissions** — agent role templates from community
