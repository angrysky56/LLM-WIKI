# Source: AGEM

The **AGEM** repository implements the Agent Group Evolving Molecular System framework.

## Key Features
- **Multi-Agent Orchestration**: Covalent/Hydrogen/Van der Waals bonding metaphors for agent interaction.
- **Sheaf-Theoretic Consensus**: Uses cellular sheaves for group-level coordination.
- **Lumpability Auditing**: Metrics for measuring information loss during context sharing.
